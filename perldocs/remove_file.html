NAME

remove_file

SYNOPSIS

remove_file

DESCRIPTION

This module supplies the methods to remove a document from an index.

METHODS

remove_doc_from_index

This method takes a doc id and the root directory of the index. It then removes the document from the index.

Ex: remove_doc_from_index($doc_id, "produced");

get_url

This is a helper method which takes in a doc id and a index root and returns the URL of the document in that index.

Ex: get_url($doc_id, "produced");

remove_doc

This method takes in a doc id in compressed form and an index root and removes the doc from the index.

Ex: remove_doc($comp_id, "produced");

remove_doc_from_tf_file

This method takes in a doc id in compressed form and a tf (term frequency) file and removes the doc id from that file.

Ex: remove_doc_from_tf_file($id, $tf_file_path);
